为什么选择 1Claw

🎴

卡片式多角色管理

每个 AI 角色一张精美卡片，图标、颜色、描述一目了然。点击卡片即刻切换对话，无需重新加载。

⚡

永久在线 · 实时对话

基于 WebSocket 长连接，所有角色常驻后台永不掉线。手机锁屏、切后台依然保持连接，随时唤醒。

🧠

Hermes 多智能体内核

每个 Profile 独立人设、独立记忆、独立会话。底层对接 Hermes Agent，支持 10+ 角色同时在线运行。

📱

原生 Flutter 跨平台

一套代码打包 iOS / Android / macOS / Windows，Native 性能、丝滑动画、完美适配手机与桌面。

🔐

自建服务 · 数据安全

Go 语言后端 + 你自己的服务器，数据不上第三方云。支持 Docker 一键部署，内网公网均可。

🎨

深色 / 浅色主题

内置暗色与亮色主题，跟随系统自动切换。极简 Metro 风格 UI，专注对话本身。
